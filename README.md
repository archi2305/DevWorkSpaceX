# Frontend Setup

## Prerequisites

- Node.js (v20 or later)
- npm
- Git

## Installation

```bash
git clone <repository-url>
cd DevWorkSpaceX/frontend
npm install
```

## Run Development Server

```bash
npm run dev
```

Open:

http://localhost:3000

## Build

```bash
npm run build
```

## Production

```bash
npm start
```
