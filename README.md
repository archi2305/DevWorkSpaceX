# ✨ Overview

DevWorkspaceX is an **AI-powered software engineering platform** that helps developers transform an idea into a complete implementation-ready software blueprint.

Instead of manually planning software architecture, databases, APIs, milestones and documentation, DevWorkspaceX leverages modern LLMs to automate the entire planning workflow.

It acts as an **AI Software Architect**, allowing developers to move from idea → blueprint → code much faster.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/yourusername/devworkspacex.git
```

```bash
cd devworkspacex
```

---

## Backend

```bash
cd backend

python -m venv venv

source venv/bin/activate

pip install -r requirements.txt

uvicorn app.main:app --reload
```

---

## Frontend

```bash
cd frontend

npm install

npm run dev
```
